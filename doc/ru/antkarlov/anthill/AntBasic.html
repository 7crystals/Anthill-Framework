activeproperty 
public var active:Boolean

Определяет активность объекта. Если объект не активен active = false, то для такого объекта не будет вызываться метод update().

The default value is true.

aliveproperty 
public var alive:Boolean

Определяет жизненый цикл объекта. Если объект "мертвый" alive = false, то это значит, что для объекта был вызван метод kill(). Для воскрешения "мертвого" объекта следует использовать метод revive().

The default value is true.

allowDebugDrawproperty 
public var allowDebugDraw:Boolean

Определяет может ли данный объект рисовать свой отладочный образ.

The default value is true.

camerasproperty

existsproperty 
public var exists:Boolean

Определеяет существование объекта. Если объект существует то для него вызываются методы update() и draw().

The default value is true.

tagproperty

visibleproperty 
public var visible:Boolean

Определяет видимость объекта. Если объект невидим visible = false, то для такого объекта не будет вызываться метод draw().

The default value is true.
