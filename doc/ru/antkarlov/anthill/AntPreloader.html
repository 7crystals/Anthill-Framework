atHome()method
protected function atHome(aHomes:Array):Boolean

Реализация элементарного сайтлока. Пример использования:

if (atHome([ "local", "mygreatsite.com" ])) {
// ура! можно играть
} else {
// нельзя играть
}

Parameters
aHomes:Array — Список доменов на которых флешке разрешено находится.

Returns
Boolean — Возвращает true если флешка находится на одном из разрешенных доменов.

getHome()method 
protected function getHome():String

Возвращает домен на котором размещена флешка.

Внимание: Если флешка размещена на домене второго или третьего уровня то вернется имя домена первого уровня, то есть если ваша игра размещена на "http://cache.armorgames.com/", то результат будет "armorgames.com".

Returns
String — Вернет local если флешка запущена на локальном компьютере или адрес домена первого уровня "domain.com".
