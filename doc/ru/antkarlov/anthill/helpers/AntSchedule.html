Classpublic class AntSchedule
InheritanceAntSchedule Object

Since : 17.01.2013

Данный класс является упрощенным менеджером задач для реализации наборов действий.

Принцип работы класса заключается в том, что в него добавляется некоторое количество методов и аргументов для них, а так же условия для экстренного прерывания выполнения исходя из изменившейся игровой обстановки.

eventCompleteproperty 
public var eventComplete:AntSignal

Событие которое генерируется при завершении всех задач в менеджере задач.

В качестве аргумента при вызове подписавшихся методов передается указатель на экземпляр класса AntSchedule.

Пример использования:

         schedule.eventComplete.add(onComplete);
         function onComplete(aSchedule:AntSchedule):void {
           trace("All tasks completed!");
         }
