beginUpdate()method 
public function beginUpdate():void

Запрещает растерезацию текста до тех пор пока не будет вызван метод endUpdate(). Следует вызывать перед тем как необходимо применить сразу много сложных операций к тексту.

setColor()method 
public function setColor(aColor:uint, aStartIndex:int = -1, aEndIndex:int = -1):void

Задает цвет текста для всего текстового поля или для указанного диапазона символов.

Примичание: Цвет указанный через setColor() применяется непосредственно к стандартному текстовому полю и не имеет отношения к значению . То есть если вы укажете цвет через setColor(), а потом зададите другой цвет через color - то цвета будут смешаны.

Parameters

aColor:uint — Цвет в который необходимо перекрасить текст.
 
aStartIndex:int (default = -1) — Начальный индекс символа скоторого начинать красить.
 
aEndIndex:int (default = -1) — Конечный индекс символа до которого красить.

updateBounds()method 
public function updateBounds(aForce:Boolean = false):void

Обновляет положение и размеры прямоугольника определяющего занимаеммую область объектом в игровом мире.

Внимание: Данный метод выполняется каждый раз перед отрисовкой объекта, но если вы изменили размеры объекта, положение объекта или положение оси объекта, то прежде чем производить какие-либо рассчеты с прямоугольником определяющего занимаемую область, необходимо вызывать данный метод вручную!

Parameters

aForce:Boolean (default = false) — Если true то положение и размеры прямоугольника будут обновлены принудительно.
